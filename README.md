## Overview
This is a lightweight, 0 dependancy notification library designed specifically for my needs. 
It is highly likely that there is a better and more customisable library out there

<br>

## Features
- Lightweight, ~1.5kb mimified
- No dependencies

<br>

## Installling
Bun
```bash
bun install https://github.com/T768D/Simple-Notif
```

npm
```
npm install https://github.com/T768D/Simple-Notif
```

<br>

## Development Requirements
- [Bun](https://bun.sh)
- [VSCode](https://code.visualstudio.com/) - Optional, launching bundler already configured
- TypeScript `bun install -g typescript`

<br>

## Bundling
- Run Select and Start Debugging
- Select Compile
